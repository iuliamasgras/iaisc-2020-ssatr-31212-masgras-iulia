Pasi de rulare ai proiectului:

1. Se importa proiectul intr-un IDE Java, de preferat IntelliJ (folosit pentru dezvoltarea proiectului) in format Maven.
2. Odata ce acesta a fost importat, din IDE se acceseaza fisierul pom.xml pentru a ne asigura ca librariile Maven au fost importate corect.
3. Se modifica fisierul `input.json` din fisierul `resources` pentru a se mapa pe Reteaua Petri dorita. Initial acesta contine Reteaua Pentri din Exemplu (lab1_exemplu_PN.jpg).
4. Se ruleaza clasa Main.java
5. Odata ce se termina rularea programului se va crea automat fisierul numit `output.txt` cu informatiile necesare.
